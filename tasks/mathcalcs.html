Математические калькуляторы

Корни квадратного уравнения ax2 + bx + c = 0

Условие: a не равно 0

Дискриминант D = b2 - 4ac

D > 0 - два корня
D = 0 - один корень
D < 0 - корней нет

Тройка Пифагора

Квадрат самого большого

числа должен быть

равен сумме квадратов

двух остальных.

Общие делители двух чисел

Общий делитель двух чисел -

это число, на которое без

остатка делятся оба числа

НОД и НОК двух чисел

НОД - наибольший общий делитель - число, на котоое делятся оба числа без остатка

НОК - наименьшее общее кратное - число, которое делится на оба числа без остатка
